## 口径说明

- `origin`: 识别到 apache/nginx/iis/openresty 但无 CDN 标志, 视为直连源站
- `unknown`: 无可识别特征 (私有 CDN / 无 server 头 / 定制网关)
- `timeout`/`error`: 本轮未能建立连接, 不代表源失效 (可能是临时网络/CDN-level 拒绝)
- 分类仅基于响应头; 同一域名可能叠加 Cloudflare→origin 链路, 本表记最外层
