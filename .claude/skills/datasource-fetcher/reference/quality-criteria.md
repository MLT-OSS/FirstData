# 数据源质量评估标准

## 6维度评分体系

每个维度按 1-5 星评分：

### 1. authority_level - 来源权威性

- **5星**: 国际顶级组织（联合国、世界银行、IMF）、国家最高统计机构
- **4星**: 知名国际组织、政府部委级机构
- **3星**: 地方政府、行业协会
- **2星**: 非官方但有一定影响力的机构
- **1星**: 个人或小型组织

### 2. methodology_transparency - 方法论透明度

- **5星**: 完整的方法论文档、详细的采集流程说明
- **4星**: 较详细的方法说明
- **3星**: 基本方法说明
- **2星**: 方法说明简略
- **1星**: 无方法说明

### 3. update_timeliness - 更新及时性

- **5星**: 实时或每日更新
- **4星**: 每周/每月更新
- **3星**: 每季度更新
- **2星**: 每年更新
- **1星**: 不定期或已停止更新

### 4. data_completeness - 数据完整性

- **5星**: 数据完整，无明显缺失
- **4星**: 数据基本完整，少量缺失
- **3星**: 有一定缺失但可用
- **2星**: 缺失较多
- **1星**: 严重不完整

### 5. documentation_quality - 文档质量

- **5星**: 详尽的文档、用户指南、API文档、示例代码
- **4星**: 较好的文档和用户指南
- **3星**: 基本文档
- **2星**: 文档简略
- **1星**: 几乎无文档

### 6. citation_count - 引用频次

- **5星**: 被学术界和政策界广泛引用（数千次以上）
- **4星**: 经常被引用（数百次）
- **3星**: 有一定引用
- **2星**: 少量引用
- **1星**: 几乎无引用

## 评分原则

1. **基于事实**: 评分必须基于实际观察，不能臆测
2. **保守评估**: 有疑问时给较低分数
3. **说明理由**: 在 assessment_notes 中说明评分依据
4. **参考现有**: 查看 sources/ 目录中类似数据源的评分作为参考
